# ToDo List CLI

A simple command-line to-do list application for managing tasks directly from your terminal.

## Features

- Create, view, and manage tasks from the CLI
- Built with C++

## Prerequisites

- CMake 3.10 or higher
- C++ compiler (GCC, Clang, or MSVC)
- Make or Ninja build system

## Building

### Clone the repository

```bash
git clone https://github.com/chasehof/ToDoListCLI.git
cd ToDoListCLI
```

### Build with CMake

```bash
mkdir build
cd build
cmake ..
make
```

The executable will be created as `ToDoList` in the `build` directory.

## Running

```bash
./build/ToDoList
```

Follow the on-screen prompts to manage your to-do list.
